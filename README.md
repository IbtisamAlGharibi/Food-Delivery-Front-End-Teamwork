🍔 Food Delivery Frontend

A responsive food delivery web application built using HTML, CSS, and Vanilla JavaScript. This project was developed as part of the Food Delivery Platform Front-End Sprint and connects to a REST API backend.

Features
Browse available restaurants
Search and filter restaurants by cuisine
View restaurant menus and add items to the cart
Place orders through the backend API
Track order status
Responsive design for desktop and mobile
Loading, empty, and error states for API requests

Technologies Used
HTML5
CSS3 (Flexbox & Grid)
JavaScript (ES6)
Fetch API
